# Disclaimer — Portfolio Case Study

This repository contains a sanitised portfolio case study based on real project work.

**The following has been anonymised:**
- Client name and organisation type have been generalised
- Company names (service provider and client) have been removed
- Contract values, financial terms, and specific personnel names have been omitted
- Proprietary data models, raw source data, and client-specific KPI logic are not included

**What is real:**
- The business problem, requirements, solution architecture, and delivery approach reflect actual work undertaken
- The activity breakdown, sprint plan, team structure, and risk register are based on genuine project artefacts
- The outcomes and metrics reflect actual results achieved on the project
